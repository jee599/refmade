Lesson 3.4 Question 7 of 12

Which pattern best describes this sequence?

2, 6, 18, 54, 162, ...

A Arithmetic (+4 each step)
B Geometric (x3 each step)
C Fibonacci (sum of previous two)
D Quadratic (n² pattern)

Correct!

Each term is multiplied by 3. This is a geometric sequence with common ratio r=3. The next term would be 162 x 3 = 486.

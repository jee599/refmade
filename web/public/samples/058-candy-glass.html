Common questions

Quick answers

Can I use it with my existing design system?

Yes. Import your Figma tokens, Tailwind config, or CSS custom properties. Candy adapts to your system — it doesn't replace it.

Is the exported code responsive?

Every export includes mobile, tablet, and desktop breakpoints. We use modern CSS (container queries, clamp()) so it scales smoothly instead of jumping between fixed sizes.

What about accessibility?

Built-in contrast checker warns you before you ship. ARIA labels auto-generated for interactive elements. We aim for WCAG AA minimum on every export.

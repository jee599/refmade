Real-time Sync

Changes propagate in under 50ms. Every keystroke, cursor move, and selection is synced instantly across all connected clients.

sync.ts
import { Flux } from '@flux/sdk'
const doc = Flux.document('project-x')
doc.on('change', (delta) => {
console.log(`${delta.user} edited`)
})

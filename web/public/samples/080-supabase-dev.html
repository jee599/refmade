Postgres Database Core

Every project is a full Postgres database, the world's most trusted relational database. With realtime functionality, database backups, extensions, and more.

import { createClient } from '@cloudbase/sdk'

const cloudbase = createClient('url', 'key')

// Fetch data with realtime subscriptions
const { data, error } = await cloudbase
  .from('posts')
  .select('*')
  .order('created_at', { ascending: false })

Authentication

Add user sign ups and logins, securing your data with Row Level Security. Supports social auth, magic links, phone auth, and more.

Edge Functions

Easily write custom code without deploying or scaling servers. Globally distributed for low latency. TypeScript by default.

Storage

Store, organize, and serve large files. From videos to images, with fine-grained policies and permissions that are easy to manage.

Realtime

Build multiplayer experiences with real-time data synchronization. Listen to database changes, broadcast, and presence.

Vector Embeddings

Integrate your favorite ML models to store, index, and search vector embeddings. Powered by pgvector for similarity search at scale.

Data APIs

Instant ready-to-use RESTful and GraphQL APIs. Auto-generated from your database schema with full type safety.
